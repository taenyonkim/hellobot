# HelloBot 통합 개발 환경 구축

## 배경

HelloBot 서비스는 9개의 독립 Git 리포지토리(서버, iOS, Android, 웹, 스튜디오, 데이터)로 구성되어 있다. 각 리포가 독립적으로 관리되다 보니, 서비스 전체를 조망하며 작업할 수 있는 통합 환경이 없었다.

Claude Code를 활용하여 여러 리포에 걸친 피쳐를 체계적으로 기획/설계/구현/검토할 수 있는 환경을 구축하고자 한다.

## 목표

1. 9개 리포지토리를 하나의 워크스페이스로 통합 관리
2. 역할별 에이전트(PM, 설계자, 개발자, 리뷰어)를 정의하여 체계적 개발 워크플로우 구축
3. 피쳐 문서를 통한 에이전트 간 커뮤니케이션 체계 확립
4. 거대한 코드베이스에서의 컨텍스트 효율성 확보

## 범위

- 포함:
  - 워크스페이스 디렉토리 구조 설계
  - CLAUDE.md 및 아키텍처 문서 작성
  - 역할별 에이전트 커맨드 9개 정의
  - 피쳐 문서 체계 및 템플릿 수립
  - 작업 가이드 문서 작성
  - 유틸리티 스크립트 (setup.sh)
- 제외:
  - 개별 리포지토리의 CLAUDE.md 수정 (별도 과업)
  - CI/CD 파이프라인 변경
  - 실제 피쳐 개발

## 2. 스킬 이용권 카드 — 기존 쿠폰 카드 형식으로 변경

현재 구현 (coopSkillVoucherItem.tsx):
- 별도 보라색 카드 (bg-[#f5f3ff], border-[#c7d2fe])
- "스킬 이용권" 배지 (bg-[#E8EAFF], text-[#6366F1])
- 상품명 + 스킬명만 표시

Figma 디자인:
- 기존 쿠폰 카드와 동일한 흰색 카드 레이아웃

변경 항목:
- [x] 카드 배경/테두리: 보라색 → **흰색 bg-white, border-[#EDEDEE]**
- [x] 라벨: "스킬 이용권" 보라 배지 → **"이용권" 태그** (흰 배경, `border-[#EDEDEE]`, `text-[#7E8185]`, 11px Bold)
- [x] 할인율 추가: **"100%"** (18px Bold, `#FF5D7A`)
- [x] 이용권 이름: 기존 유지 (16px Bold, `#242526`)
- [x] 챗봇명: 미노출 (디자인에서 opacity: 0)
- [x] 하단 좌측: **만료일 추가** "YYYY.MM.DD HH:mm까지" (12px Regular, `#7E8185`)
- [x] 하단 좌측: **만료 임박 표시** "N일 남음" (12px Regular, `#F23658`) + 구분선
- [x] 하단 우측: **"스킬 보러가기 >"** 링크 추가 (12px Bold, `#BE7AFE`)
- [x] 카드 탭 → 스킬 상세 페이지 이동 동작 확인

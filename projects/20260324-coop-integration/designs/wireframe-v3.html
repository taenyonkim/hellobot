쿠폰 화면

진입: 프로필 → 쿠폰 탭, 또는 카카오 선물하기 딥링크로 직접 랜딩
기존 쿠폰 입력란을 그대로 사용. 쿠폰 코드 입력 → 등록
서버가 코드 형식을 판별하여 기존 쿠폰이면 기존 처리, 카카오 상품권이면 조회 API → 상품유형별 확인 팝업
화면 유형: 네이티브 (iOS / Android / Web)
입력란 하단 힌트: "헬로우봇 쿠폰, 카카오 상품권 등 쿠폰 번호를 입력해보세요."
입력 필드 비어있으면 등록 버튼 비활성화

하트 충전권 확인 팝업

조회 성공 + productType = heart
상품 이미지 + 상품명 + 충전 수량 표시
"이 쿠폰을 사용하면 하트 N개가 충전됩니다. 사용하시겠습니까?"
사용하기 → L1 사용 API → 성공 시 충전 완료 팝업
중복 탭 방지: 사용하기 버튼 로딩 + 비활성화
취소 → 팝업 닫기, 쿠폰 미사용

스킬 교환권 확인 팝업

조회 성공 + productType = skill
스킬 배너 이미지 + 스킬명 + 챗봇명 표시
"이 쿠폰을 스킬 이용권으로 교환합니다. 사용하시겠습니까?"
사용하기 → L1 사용 API → 쿠폰 리스트에 스킬 이용권 등록
중복 탭 방지: 사용하기 버튼 로딩 + 비활성화

하트 충전 완료

"하트 {N}개가 충전되었어요"
충전 확인하기 → 내 하트 페이지로 이동하여 충전 결과 확인

내 하트 페이지

하트 충전 완료 후 충전 확인하기 탭 시 이동하는 화면
보유 하트 수량이 충전분 반영되어 표시 (4,536 → 5,036)
충전 내역 최상단에 "카카오 상품권 하트 충전" 항목 노출
기존 하트 페이지 화면 그대로 활용

스킬 이용권 등록 완료

사용하기 → 팝업 닫힘 → 쿠폰 리스트 최상단에 스킬 이용권 카드 추가
상단 토스트: "스킬 이용권이 등록되었어요"
이용권 카드에 스킬명, 챗봇명, 유효기간 표시
이용권 카드 탭 → 스킬 상세 페이지로 이동 (쿠폰 적용, 가격 0원)

스킬 상세 페이지

쿠폰 리스트에서 스킬 이용권 카드 탭 시 해당 스킬 상세 페이지로 이동
기존 스킬 상세 페이지와 동일한 UI
쿠폰 적용으로 가격이 ♥ 0으로 표시 (원래 가격 취소선)
지금 보러가기 → 챗봇 채팅방으로 이동, 쿠폰 자동 적용되어 무료 이용

챗봇 채팅방

스킬 이용권의 지금 보러가기 탭 시 이동하는 화면
해당 챗봇의 채팅방에서 스킬이 자동 시작됨
스킬 이용권이 적용되어 별도 하트 차감 없이 진행
기존 채팅방 UI 그대로 활용

## 1. 기존 분석과의 정합성 검토 — 지지 입장 (For)

### F1. **v2 M2 (즉시 이해도) ★★★ 가중치와 직접 일치**

[`skill-priority-analysis-v2 §1`](./skill-priority-analysis-v2-market-fit.md#1-새-분석-프레임-4축):
> M2 즉시 이해도 — 상품명·섬네일만 보고 5초 안에 "이게 뭐고 누구에게 줄지" 떠오르는가. 가중치 ★★★.

→ 보편 콘텐츠 = M2 ◎ 보장. 디테일·미시 콘텐츠 = M2 △ 위험. **사용자 가설 ≡ M2 우선 채택**.

### F2. **v2 1차 라인업 8/10이 이미 보편 콘텐츠**

### F3. **카카오 운세·부적 카테고리 79.3% = 보편 트리거 (리포트C)**

- 명태 액막이 (이사·집들이·새차) / 오행 / 신년 = 모두 보편 라이프이벤트 트리거
- 보편 콘텐츠 = 카카오 운세 시장에서 **이미 검증된 행동 패턴**
- 디테일·미시 콘텐츠는 카카오 매대에 거의 없음 — 새 시장 개척 비용 큼

### F4. **A.7.2 헬봇 내부 실측 — 26-40 사주 77~82%**

[skill-launch-roadmap §0.A.7.2](./skill-launch-roadmap.md#a7-참고5-헬봇-앱-내부-세그먼트-eda--우리-사용자-실측-app-only-30일):
- 결제 코어층(26-40)도 사주 콘텐츠가 압도 (77~82%)
- topic 매출: 연애 76% + 총운 13% + 결혼 10% = **보편 카테고리만으로 99% 매출 점유** (data/topic-by-age-12m.csv)
- 미시·디테일 콘텐츠("기타·자기탐구")는 매출의 1% 미만

→ **보편 콘텐츠가 헬봇 매출의 99% 정의** — 카카오 진입 우선순위 데이터 정합.

### F5. **D5 (수신 안전성) 양극화 회피의 직접 해법**

[`02-sender-persona-hypothesis §1 D5`](./02-sender-persona-hypothesis.md#d5가-가장-결정적):
> 운세는 "안전한 default 선물"이 아니라 "정확히 꽂는 선물".

보편 콘텐츠는:
- 받는 사람 운세 친화도 낮아도 **콘텐츠 자체의 보편성으로 정당화** (결혼 = 궁합, 새해 = 신년운세)
- 친구 카톡에 "[궁합] 보고서가 도착했어요" → 부담 낮음
- 디테일 콘텐츠 "[이별 후 재회 가능?]" → 받는 사람 사정에 따라 부담 큼

→ 보편 콘텐츠는 **D5 양극화를 가장 좁히는** 라인업 전략.

### F6. **P5 (디스커버리·셀프) 진입 SKU에 가장 친화**

[`02-sender-persona-hypothesis P5`](./02-sender-persona-hypothesis.md#p5-디스커버리--셀프-진입형--p1p4와-다른-축-신규-유입-채널의-본질):
- P5 = 헬봇 비사용자가 매대에서 발견 후 본인 호기심 발동
- 첫 운세 결제는 **이름 들어본 콘텐츠 = 보편**부터 시작
- "사주팔자", "궁합", "신년운세" 같은 키워드는 헬봇 모르는 사람도 즉시 의미 해독

→ P5 신규 유입 채널 가치를 극대화하려면 보편 콘텐츠 라인업 필수.

### F7. **M4 (코어 팬 매력 유지)도 동시 충족**

보편 콘텐츠도 풀이 깊이가 정통이면 코어 팬 만족 — 결혼 궁합·신년운세는 마니아도 매년 봄. 보편 ≠ 가벼움. **보편 + 정통**이 매대 노출용으로 가장 안전한 조합.

## 2. 반대 입장 (Against / Devil's Advocate)

같은 결론을 다른 각도에서 흔드는 7가지 반론. 토론을 풍부하게 하기 위해 의도적으로 강하게 작성.

### A1. **보편 콘텐츠는 차별화가 약해 헬봇만의 메시지 부재**

- "신년운세" = 헬봇 외에도 점쟁이·잡지·다른 앱·미디어·운세 콘텐츠 다 함. **명백한 commodity**
- 카카오 매대에서 "왜 헬봇 신년운세?"에 답이 없으면 매대 알고리즘·MD 가산점 약함
- 헬봇의 인앱 강점은 **페르소나형 카피·미시 디테일** ("운빨충에게", "이 사람과 재회 가능?")
- 이 강점을 카카오로 옮겨야 차별화 ↔ 보편 콘텐츠는 강점 희석

`[반론]` 보편으로 진입하면 헬봇 = 일반 운세 앱으로 인지 → **First Mover 효과 약화**

### A2. **보편 콘텐츠는 시즌·맥락 의존성 강함 → 평시 매출 절벽**

- 신년운세 = 11~2월만 활성 (헬봇 12M 매출 1위 KS00001 = 6M에 63% 집중, 30D는 비시즌 급락)
- 결혼 궁합 = 결혼 트리거 있는 사람만 (사용자 풀 좁음)
- 합격 = 수능·취업 시즌만
- → 보편 라인업은 **시즌 외(6~8월) 평시 매출 절벽** 위험
- 디테일·일상 콘텐츠 (오늘 운세·연애 단발·일진) 가 시즌 무관 일상 결제 흐름 만들 수 있음

`[반론]` exposure-strategy §3.4 D+90 게이트 (월 거래액 500만원) 평시 통과 가능한가? 보편만으로는 매대 매출 시계열 불안정.

### A3. **보편 콘텐츠 = 자사몰 자기잠식 직접 위험**

- 헬봇 인앱 매출 TOP 1 신년운세, TOP 3~10 모두 보편 카테고리 (data/skills-by-segment-12m.csv 확인)
- 카카오에 같은 보편 SKU 진열 → 인앱 결제자가 카카오로 채널 이동
- 카카오 수수료 8% → 매출 총량은 카니발리제이션 비율만큼 감소
- 카카오 단독·차별 SKU가 자기잠식 회피의 정석

`[반론]` 보편 위주 라인업은 **카카오 매출 ↑ but 자사몰 매출 ↓** 동시 발생 위험. exposure-strategy §7.1 F1 함정 직격.

### A4. **보편 콘텐츠는 P3·P5 호기심 트리거 약함**

- P3 (재미·놀이) 페르소나 → 단톡방 트리거. "[신년운세 보고서]"보다 "[이상형 스포]·[연애 존잘 존예]" 가 단톡 공유 가능성 ↑
- P5 (디스커버리·셀프) → 매대 광고 효과. 호기심 트리거가 클릭률(CTR) 결정. "신년운세"는 진중·예측 가능, **"내 운명의 상대 외모는?"** 같은 자극적 제목이 CTR ↑
- 카카오 매대 클릭률 → 매대 알고리즘 가중치 → 노출 빈도 → 매출
- 보편 위주 라인업은 **CTR 최적화 실패 가능**

`[반론]` 매출보다 노출·CTR을 잡는 게 First Mover 게임 — 차별·자극이 노출 게임의 무기.

### A5. **보편 = 가격 천장 낮음 → C2 Whale 흡수 실패**

- 신년운세 ₩37,500 / 결혼 궁합 ₩19,500 / 일반 사주 ₩17,000~22,500
- C2 (36-40) 30K+ Whale 5.4% — 정통 디테일 패키지(창광 ₩60,000 류)가 진짜 흡수처
- 보편 위주는 ARPPU 천장 약 ₩30,000 → C2 Whale 매출 손실
- 디테일 프리미엄 콘텐츠가 ARPPU 게임 무기

`[반론]` 보편 라인업은 **객단가 천장이 낮아 매출 확장 한계**. Phase 2~3 매출 50% 추가 도달 어려움.

### A6. **보편 = 후발자 진입 쉬움 → First Mover 효과 약화**

- 다른 운세 앱·점성술 콘텐츠 회사가 카카오에 진입하면 **보편 콘텐츠로 똑같이 진입**
- 헬봇이 First Mover라도 보편 시장은 진입 장벽 낮음
- 디테일·차별 콘텐츠로 카테고리 정의해야 "헬봇 = 운세 = 디테일 정통 사주" 등치 가능
- 후발자가 못 따라하는 깊이가 보호막

`[반론]` First Mover 가산점은 차별 콘텐츠에서 가장 큼. 보편 진입은 First Mover 가산점 잠재력 작음.

### A7. **보편 콘텐츠는 콘텐츠 R&D 압력장 가치 없음**

[`exposure-strategy §1.4`](./exposure-strategy.md#14-4차-역할-콘텐츠-rd-압력장):
> 카카오 진입은 **본 서비스의 콘텐츠·UX 품질을 끌어올리는 강제력**.

- 보편 콘텐츠는 이미 인앱에 있음 → 신규 기획 압력 없음
- 차별·신규 콘텐츠를 카카오 전용으로 기획하면 R&D 자극 + 인앱 역수출 가능
- 보편 라인업은 **카카오 진입의 4차 역할(R&D 압력장)을 거의 비활성화**

`[반론]` 단순 자산 이식이 아니라 콘텐츠 R&D 트리거로 활용하려면 차별 SKU 일부 포함 필수.

## 3. 합성 의견 — 보편 100% vs 보편 + 차별 혼합

지지(F1~F7) + 반대(A1~A7) 양쪽을 모두 흡수하는 합리적 합성:

### Phase 1 (출시 ~ D+90)
- **보편 콘텐츠 80% (8~10 SKU)** — F1·F2·F3·F4·F5·F6·F7 모두 ◎. 신규 진입 안정성 최우선.
- **차별 콘텐츠 10% (1~2 SKU)** — A1·A4·A6 부분 흡수. 카카오 단독 차별 SKU 1개 시범 (예: "사주 직업 컨설팅" 인앱과 다른 카피·패키지)
- **라이트·자극 콘텐츠 10% (1~2 SKU)** — A4 P3·P5 CTR 흡수. ₩9,900 라이트 SKU (KS00011 데이트뷰 류, 단 메시지 톤 분리)

### Phase 2 (D+90 ~ D+270)
- **차별 비중 ↑ 25~30%** — Phase 1 성과 학습 후 R&D 트리거. C2 Whale 흡수용 프리미엄 SKU 1~2개 신설
- 시즌 부스트 확장 (어버이날·수능·발렌타인)

### Phase 3 (D+270 ~ D+360+)
- **차별·디테일 라인 본격 확장 40%+** — A1·A6 흡수. "헬봇 = 정통 디테일 사주" 카테고리 정의
- 보편 라인은 베이스라인 유지

### 비유
> **보편은 매대 매출의 "베이스라인"이고, 차별은 매대 매출의 "차별점"**. Phase 1은 베이스라인 우선, Phase 2~3은 차별점 확장.

## 4. 풀어야 할 진짜 토론 분기 (이 메모의 핵심)

지지 vs 반대 중 **실증적 검증**이 필요한 3가지:

### Q1. **자기잠식 비율 (A3 vs F1·F2)**
- 보편 라인업이 자사몰 매출을 얼마나 잠식할 것인가?
- 0% → 보편 100% 안전. 30% → 차별 SKU 비중 늘려야.
- 검증: Phase 1 D+30~D+60 동안 인앱 신년운세 매출 추이 카카오 등록 코호트 vs 비등록 코호트 비교

### Q2. **CTR 차이 (A4 vs F6)**
- 보편 SKU vs 자극·차별 SKU의 카카오 매대 CTR 격차는 얼마나 큰가?
- 매대 알고리즘 가중치에 미치는 영향 직접 측정 필요
- 검증: Phase 1 A/B — 같은 매대에 보편 SKU + 차별 SKU 동시 노출, CTR·구매 전환율 비교

### Q3. **평시 매출 절벽 (A2 vs F3)**
- 보편 위주 라인업이 시즌 외 평시(6~8월) 매출을 얼마나 유지할 수 있나?
- 운세·부적 카테고리 시즌성이 보편 위주에서도 평시 매출 베이스라인 만드는가?
- 검증: 카카오 운세·부적 카테고리 월별 매출 시계열 분석 (리포트C 확장, 카카오 MD 협조)

## 5. PM 1차 입장 (최종)

> **사용자 가설을 Phase 1에 강하게 채택. Phase 2부터는 차별 비중 확장으로 양쪽 흡수.**

이유:
1. 7개 분석 축(F1~F7)이 모두 보편 우선을 지지 — 의사결정 신뢰도 매우 높음
2. **신규 진입 첫 90일의 가장 큰 리스크는 D5 양극화** (받는 사람이 운세 안 좋아함 → 미사용·환불·매대 가중치 하락). 보편 콘텐츠가 이 리스크 회피의 정석
3. 차별·디테일 콘텐츠는 학습된 매대에서 더 큰 효과 — Phase 1에 차별로 시작하면 매대 알고리즘 학습 부족 + 실패 시 회복 어려움
4. 자사몰 자기잠식 우려(A3)는 카카오 단독 SKU 1~2개로 1차 회피 가능
5. C2 Whale 흡수(A5)는 Phase 2 어버이날·환갑·정통 패키지 SKU에서 해결

### 사용자 가설을 그대로 채택 시 라인업 (1차 14 SKU 가이드)

| Tier | 가격대 | SKU 수 | 보편 vs 차별 |
|---|---|:-:|---|
| **시즌 보편 핵심** | ₩29,900~₩37,500 | 2 | KS00001 신년 / KS00029 학부모 — 시즌 ON/OFF |
| **메인 보편** | ₩17,000~₩22,500 | 6 | 결혼 궁합 / 직업 컨설팅 / 이사 풍수 / 합격·응원 / 부부 궁합 / 범용 대운 — 모두 보편 |
| **차별 시범** | ₩17,000~₩19,500 | 1~2 | 카카오 단독 SKU (자사몰 카피·구성과 차별화) |
| **라이트·자극** | ₩9,900 | 2~3 | KS00011 데이트뷰 · KS00015 평생 재물 · 라이트 타로 1개 (P3·P5 진입 게이트) |
| **프리미엄 (Phase 1 보류)** | ₩29,900~₩49,900 | 0 | Phase 2에 도입 (C2 Whale 흡수, A5 흡수) |

→ **14 SKU 중 보편 8 + 차별 1~2 + 라이트 2~3 + 시즌 2 = 약 85% 보편 비중**. 사용자 가설의 의도와 일치.

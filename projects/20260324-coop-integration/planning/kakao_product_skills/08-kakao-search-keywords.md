### 0.1 카카오 선물하기가 제공하지 않는 데이터

| 데이터 | 공개 여부 | 영향 |

### 0.3 카카오 키워드별 시장 인식 강도 (P1 재정리)

| 키워드 | 노출 상품 수 | 주된 상품 유형 | 선물 매력 | 헬봇 적합도 평가 |
|---|---:|---|:-:|---|
| **운세** | 87 | NFC 키링 (활성, 후기 多) | ★★★★★ | **운세 콘텐츠 핵심 어휘** — 캐주얼·라이트 |
| **부적** | 다수 | NFC 키링·오마모리 (활성) | ★★★★★ | 캐주얼·합격·응원 시그널 — 굿즈 트랙 |
| **타로** | 304 | 타로 카드덱 + 책 | ★★★★ | 16-25·20대 진입 어휘 |
| **신년운세** | 377 | 책 + 키링·부적 + 선물세트 | ★★★ | **시즌 핵심** (11~2월 부스트) |
| **사주풀이** | 1,588 | 95% 책 | ★★ | 책 카테고리 dominant — 디지털 콘텐츠 충돌 |
| **토정비결** | 615 | 100% 책 | ★ | 시즌 키워드 후보 (말띠 해 2026 운세) |
| **사주** | 463 | 95% 책 | ★★ | "사주" 단독은 책 카테고리에 밀림 — 변형 필요 |
| **궁합** | 98 | 100% 책 | ★★ | 책 외 활성 굿즈 없음 — Blue Ocean |
| **사주이용권** | 912 | **무관** (디즈니+·멜론) | (N/A) | ⚠️ **검색 의도 불일치** — "이용권" 단어를 핵심 키워드로 쓰지 말 것 |
| 헬로우봇 | 0 | 무관 폴백 | — | 브랜드 키워드 미형성 — 의존 불가 |
| 포스텔러 | 0 | 무관 폴백 | — | 운세 앱 브랜드 자체가 카카오에 없음 |

**핵심 발견 (P1)**:
- **"운세" / "부적" / "타로"**는 ★★★★★급 활성 시장이지만 **굿즈 영역**. 디지털 콘텐츠로 진입하면 검색 결과 상단 노출은 어려우나 **매대 동승 노출**(굿즈 옆에 운세 이용권)에서 자연스러움.
- **"사주풀이" / "토정비결" / "사주"**는 노출량 크지만 **책에 95% 점유** — 헬봇 디지털 콘텐츠가 책 검색 결과에서 발견되기 어려움. **수식어 결합 변형 필수** (예: "사주 풀이 이용권" 단순 키워드보다 "친구에게 보내는 사주 운세" 같은 상황 결합형).
- **"사주이용권"은 함정 키워드** — 사용자가 이 단어를 사주 콘텐츠 검색에 사용하지 않음 (현재 노출 결과는 디즈니+/멜론). **"이용권" 단독을 상품명 핵심에 두면 SOV 0**.
- **"신년운세"는 시즌 핵심** — 11~2월 노출 트래픽 흡수 가능. 비시즌에는 약함.

## 1. 키워드 도출 프레임워크

### 1.1 키워드 5층 분류

상품명에 임베드할 키워드를 5층으로 분류 — 한 상품명에 **층별 1개씩 합성**하여 매칭 폭을 넓힘.

| 층 | 정의 | 예시 | 검증 출처 |
|:-:|---|---|---|
| **K1. 시장 키워드** | 카카오에서 이미 검색되는 일반 어휘 | 운세 / 사주 / 신년운세 / 궁합 / 타로 / 부적 | P1 노출량 |
| **K2. 상황 키워드** | 카카오 9개 매대 + 14개 선물테마 | 신년 / 결혼 / 이직 / 입학 / 어버이날 / 이사 / 집들이 / 합격 / 응원 / 생일 / 솔로 | P3 매대 |
| **K3. 관계 키워드** | 받는 사람 라벨 | 친구 / 베프 / 단톡 / 솔로 / 직장 동료 / 후배 / 동기 / 신혼 부부 / 학부모 | 02 페르소나 |
| **K4. 콘텐츠 정체성** | 사용자가 받을 콘텐츠가 무엇인지 명확화 | 사주 풀이 / 타로 리딩 / 운세 보고서 / 궁합 분석 / 사주 이용권 | 04 §3.1 |
| **K5. 혜택·톤 키워드** | 왜 보내는지·받으면 무엇이 좋은지 | 행운 / 응원 / 위로 / 축하 / 시작 / 한 해 가이드 / 선물 | 04 §3.1 |

### 1.2 상품명 합성 패턴 (권장)

```
[K2 상황] + [K3 관계] + [K4 콘텐츠] + [K5 혜택] (+ [K1 시장] 필요시)

예: "결혼을 축하해 — 신혼 부부 사주 궁합 분석 (결혼 선물 운세 이용권)"
       └ K2: 결혼 축하    └ K3: 신혼 부부  └ K4: 사주 궁합 분석  └ K5: 결혼 선물   └ K1: 운세 이용권
```

**규칙**:
- 한 상품명에 K1·K2·K4 **최소 3개층 필수** (검색 매칭 보장)
- K3·K5는 SKU 페르소나에 맞춰 1개씩 추가
- 카카오 상품명 글자수 제한 (대략 40~60자 권장)
- "이용권"은 K4의 보조 키워드로만 — **단독 사용 금지** (P1 §0.3 함정 키워드 분석)
- 신년 시즌 SKU는 **K1=신년운세 필수** (377 노출 시장 흡수)
- 비시즌 SKU는 **K1=운세 + K2 상황**로 ★★★★★ 활성 시장 옆에 자리 차지

## 6. 한계 및 후속

### 6.1 한계

- **카카오 검색량 비공개 — 본 분석은 프록시 데이터 추정**. 정량 우선순위는 출시 후 V6·V7로 검증 보정 필요.
- **카카오 상품명 매칭 알고리즘 미공개** — "동의어 매칭 / 부분 매칭 / 토큰화 단위"가 불명. K1·K2·K4 4층 임베드로 매칭 폭을 넓히는 것이 안전 전략이지만, 카카오가 첫 단어 가중치를 강하게 줄 수도 있음 — **첫 4~5어절에 K1+K2 배치 권장**.
- **글자수 제한** — 카카오 상품명은 대략 60자 안팎. 권장 템플릿이 모두 60자 안에 들어가는지 V1 검증 필요.
- **헬봇 인앱 어휘 ≠ 카카오 발신 어휘** — 인앱 매출 코어 단어("천년배필", "팔자")는 카카오 비유저에게는 낯설 수 있음. K4 정체성 키워드로 보완.

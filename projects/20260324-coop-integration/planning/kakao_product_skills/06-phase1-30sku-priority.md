## 0. 한 줄 요약

> Phase 1 = **연령 무관 코어 SKU 14개 (1순위) + 26-50 공통 보편 10개 (2순위) + 16-25 라이트 6개 (3순위) = 30개**.
> 핵심 발견: **연령대별 매출 TOP10이 거의 동일한 SKU 풀** — 헬봇 인앱은 26-35·36-50·16-25 모두 같은 SKU에서 결제. 라인업 선정도 **연령별 분리가 아니라 단일 풀에서 우선순위**.

## 1. 데이터 백업 — 연령대별 인앱 지표 (12M, 현재 판매중)

### 1.1 연령대 전체 현황

| 연령대 | 활성 SKU | 구매자 합 | 매출 합 | 매출 비중 | 구매자 비중 | 평균 ARPPU |
|---|---:|---:|---:|---:|---:|---:|
| **26-35** ★ | 329 | **154,374** | **23.12억** | **59%** | **54%** | 14,978 |

**관찰**:
- **26-35 매출 59% / 구매자 54%** — 매출 비중 > 구매자 비중 (단가 높은 SKU 결제 패턴)
- **16-25 매출 26% / 구매자 33%** — 구매자 비중 > 매출 비중 (라이트 SKU 결제 패턴, ARPPU 11K)
- **36-50 매출 15% / 구매자 13%** — 작지만 ARPPU 가장 높음 (15,898) → C2 Whale 흡수

### 1.3 16-25 라이트 SKU 풀 (S1) — 3순위 근거

16-25 구매자수 TOP에는 ₩1,500 라이트가 다수:
- 2141 그 사람의 속마음 (5,695명, ₩1,500)
- 400 다시 만날 수 있을까 (3,243명, ₩1,500)
- 2505 지금 날 좋아하는 사람은? (2,217명, ₩1,500)
- 13663 어떤 사람과 사귈 운명? (1,888명, ₩1,500)
- 1210 우리 얼마나 오래 갈까? (1,681명, ₩1,500)

→ **16-25는 ₩1,500 라이트 SKU 반복 결제 패턴**. 단가 작아도 구매자 풀 매우 큼. 카카오 진입 가능성은 ₩1,500 X (카카오 최저가 정책 검토 필요) → **₩6,000~₩9,000 라이트 SKU 또는 묶음으로 진입**.

## 4. 시기별 출시 진행 가이드

### Phase 1-A (D-day) — 1순위 14 SKU
- **목표**: 매대 매출 베이스라인 안정. 26-35 코어 흡수.
- **노출 매대**: 일반 매대 + 친구 생일 매대 + 신년 매대 (시즌 ON) + 결혼 매대
- **D+7 게이트**: 일평균 등록 ≥ 20건, 성공률 ≥ 99%, 신규 구매자 비율 ≥ 50%
- **D+30 게이트**: K값 ≥ 0.2, 14 SKU 중 10개 이상 매주 1+ 등록, **자기잠식 비율 측정 1차 보고**

### Phase 1-B (D+30) — 2순위 10 SKU 추가 (총 24)
- **목표**: 매대 폭 확장. 36-50 Whale 흡수. 시즌 부스트 캘린더 진입.
- **노출 매대**: 이사·집들이 / 어버이날·환갑(시즌) / 자녀 입학 / 임신·출산 축하
- **시기 동기**: 어버이날(5월) / 임신 트리거 / 학부모 시즌
- **D+60 게이트**: 라인업 확장 후 매대 가중치 ↑ 확인, ARPPU ↑ 확인

### Phase 1-C (D+60) — 3순위 6 SKU 추가 (총 30)
- **목표**: 구매자 풀 확장. K값 엔진 활성화. P3·P5 진입 게이트.
- **노출 매대**: 친구 생일 라이트 / 발렌타인·화이트데이 시즌(2~3월)
- **D+90 최종 게이트**: K값 ≥ 0.3, 월 거래액 ≥ 500만원 (exposure-strategy §3.4)

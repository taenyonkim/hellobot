|:-:|---|---|---|---|---|
| **S1** | 16-25 | P3 (재미) + P5 진입 | 연애·일반운세 (라이트) | ≤ ₩12,000 | 단톡방·위시리스트 |

| **S3** | 26-35 | P1 (자가소비→권유) | 연애·일반운세·총운 | ₩17K~29.9K | **코어 결제자, 친구 발신** |
| **S4** | 26-35 | P2 (라이프이벤트) | 결혼·학업직업·재물금전·자기탐구 | ₩17K~37.5K | 결혼·이직·이사 트리거 |

| **S6** | 36-50 | P4 (정통·프리미엄) | 총운·신년(프리미엄) | ≥ ₩29,900 | Whale |

### SKU → 세그먼트 라벨링 규칙

```
primary = argmax(c3 = age_16_20 + age_21_25,
                 c1 = age_26_30 + age_31_35,
                 c2 = age_36_40 + age_41_45 + age_46_50)

primary == 0 (결제자 없음) → Other
primary == c3 (16-25):
  if price ≤ 12,000 → S1
  else → S2
primary == c1 (26-35):
  if topic ∈ {결혼·학업직업·재물금전·자기탐구} → S4
  else → S3
primary == c2 (36-50):
  if price ≥ 29,900 → S6
  else → S5
```

## 2. 핵심 통찰

### 2.1 S3 (26-35 자가소비) 압도적 코어 — 카카오 1차 라인업의 핵심

- 매출 점유 82.6% (SKU 라벨링) / 51% (BQ 연령 매트릭스)
- 매출 TOP 3 모두 S3:
  - KS00001 신년운세 보고서 ₩37,500 (3.38억)
  - 천년배필 ₩21,000 (1.69억)
  - 솔로 탈출 시기 ₩24,000 (1.49억)
- **카카오 1차 라인업 14 SKU 중 8~10개는 S3에서 선정** (04번 결정 §1.1 라인업 가이드 정합)

### 2.2 ⚠ SKU 라벨링 vs 결제자 분포는 다른 차원

| 데이터 | 36-50 비중 | 차이 |
|---|---:|---|
| BQ 연령 매트릭스 매출 | **14%** (7.5억) | 36-50 사용자 결제 매출 |
| SKU 라벨링 매출 (S5+S6) | **0.88%** (0.41억) | 36-50 primary SKU 매출 |

**차이 = 7.0억** — **36-50 사용자가 26-35 primary SKU를 결제**하기 때문.

즉:
- 36-50 사용자도 결혼 궁합·연애 사주·신년운세를 결제 (26-35와 동일 SKU)
- 라벨링상 "26-35 SKU"이지만 36-50도 잘 결제하는 SKU 다수

**카카오 1차 라인업 함의**:
- S5·S6 SKU만 골라서 36-50 매대에 진입하면 매출 잠재력 작음
- 36-50 발신자(P2 가족·라이프이벤트)는 **S3·S4 SKU에서 가족 응원·자기 보상으로 발신** 가능성 큼
- 라인업 14 SKU에 S5·S6은 1개씩만 시드 충분, **S3 SKU 카피·매대를 36-50 친화로 변형**하는 게 효율적

### 2.3 16-25 매출은 라이트(S1) > 자가소비(S2)

- S1 (16-25 라이트, 가격 ≤ ₩12K): 매출 1.53억 / SKU 435개 / 구매자 61,905명 (1.5만원 평균)
- S2 (16-25 자가소비, 가격 > ₩12K): 매출 0.40억 / SKU 197개 / 구매자 2,743명

→ **16-25는 라이트 가격대(₩9,900)에서 활발**, 단가 ₩17K+ 영역에서는 결제 적음. 04번 결정 §1.1 "C3 라이트 진입 SKU 3개"의 데이터 근거.

### 2.4 가족자녀 ARPPU 최고, 그러나 모집단 작음

- 가족자녀 26-35: 2,715명 / 0.94억 / ARPPU 34,659원 (S3에 흡수됨)
- 가족자녀 36-50: 904명 / 0.31억 / ARPPU 33,914원 (S5에 흡수됨)
- → KS00029 "학부모 자녀 사주" 카카오 진입의 정량 근거. 단 모집단 작음 → **시즌 부스트(1~3월 입학 시즌) 운영 필수**

### 2.5 가설 보정 — 02번 메모 페르소나 비중

02번 메모의 행위 비중 가설(P1 45% / P2 22% / P3 13% / P4 5% / P5 10%)을 헬봇 인앱 매출 라벨링 데이터로 보정:

| 페르소나 | 02번 가설 | 인앱 SKU 라벨 비중 | 차이 |
|---|---:|---:|---|
| P1 (자가소비 권유) ≈ S3 | 45% | **82.6%** | 헬봇 인앱에서는 P1 코어 압도적 |
| P2 (라이프이벤트) ≈ S4 + S5 | 22% | **13.7%** (S4 12.9 + S5 0.8) | 헬봇 인앱 가설보다 작음 |
| P3 (재미·놀이) ≈ S1 + S2 | 13% | **3.9%** (S1 3.1 + S2 0.8) | 헬봇 인앱 가설보다 작음 |
| P4 (정통) ≈ S6 | 5% | **0.08%** | 거의 없음 — 인앱 자체에 정통 프리미엄 SKU 미발달 |
| P5 (디스커버리) | 10% | 측정 불가 | 헬봇 비사용자라 인앱 데이터에 안 잡힘 |

→ **카카오는 헬봇과 다른 게임**:
- 헬봇 인앱: P1 자가소비 82.6% 압도
- 카카오 발신: P2 라이프이벤트·P3 재미·P5 디스커버리 비중이 더 클 가능성
- **헬봇 코어 SKU(S3)만 카카오에 옮기면 P2·P3·P5 매대 매출 손실 가능**

### 4.2 카카오 매대 매출 잠재력 추정 (헬봇 인앱 기준)

가정: 카카오 매대 노출 매출 ≈ 헬봇 인앱 12M 매출의 5~15% (보수~기준)

| Tier | 인앱 12M 매출 합 | 카카오 잠재 (10% 기준) |
|---|---:|---:|
| 시즌 핵심 2 SKU | 3.38억 + 추정 0.5억 = 3.88억 | 3,880만 |
| 메인 보편 6 SKU (S3 TOP 6) | 약 9.5억 | 9,500만 |
| 라이프이벤트 2 SKU (S4 TOP 2) | 약 1.2억 | 1,200만 |

| C2 시즌 1 SKU (S5 TOP 1) | 0.06억 | 60만 |
| **합** | **15.3억** | **1.5억** |

→ 카카오 첫 90일 매출 잠재력 약 1.5억 (12M 기준 환산 시 분기 3,800만 / 월 1,260만). exposure-strategy §3.4 D+90 게이트(월 500만) 의 2.5배 안전 마진. 단 자기잠식 비율(R6)을 빼면 실제 추가 매출은 더 작을 수 있음.

### 4.3 R6 자기잠식 측정 인프라의 데이터 출발점

각 SKU의 12M 인앱 매출이 04번 §4.1 자기잠식 비율 측정의 베이스라인:
- 잠식 비율 = 카카오 D+30 매출 ÷ (카카오 D+30 매출 + 인앱 D+30 동일 SKU 매출)
- 또는 = (베이스라인 인앱 매출 - D+30 인앱 매출) ÷ 베이스라인

→ `/dev-data` 카카오 등록자 코호트 마트 + skills-by-segment 12M 매출을 비교 기준으로 활용.

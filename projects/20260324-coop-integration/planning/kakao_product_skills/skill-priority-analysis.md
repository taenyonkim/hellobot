## 1. 분석 프레임워크

카카오 선물하기 사용자의 쇼핑 동선(이벤트 발생 → 받는 사람·상황 결정 → 탐색 → 선택)을 따라가면, **선물용 스킬이 충족해야 하는 조건**은 4가지로 압축된다.

| # | 평가축 | 정의 | 가중치 |
|---|--------|------|--------|
| A | **선물 컨텍스트 명료성** | 구매자가 "이 상황에 이걸 줘야지"라고 즉시 떠올릴 수 있는가 (탐색 동선에서 잡히는가) | ★★★ |
| B | **수용 부담 ↓** | 받는 사람이 본인의 정체성/약점을 노출하지 않고 받을 수 있는가 (예: "솔로 탈출"은 받는 즉시 상태 인정 강요) | ★★★ |
| C | **가치 체감** | 받은 사람이 "고마워 / 도움됐어 / 힘이 됐어 / 재미있었어" 중 하나를 명확히 느끼는가 | ★★ |
| D | **이벤트 빈도** | 한국 사회에서 그 트리거 이벤트가 1년에 몇 번이나 발생하는가 (생일 > 결혼/이직 > 출산/이사 > 입학 > 합격) | ★★ |

**B(수용 부담 ↓)를 ★★★로 잡은 이유**: 카카오 선물하기는 받는 사람이 "헐 너 이거 봤어?" 하고 캡쳐해서 친구한테 보내는 노출형 선물 채널. 받는 사람이 부담을 느끼면 즉시 환불 또는 미사용 → 가치 체감 0. "솔로 탈출 시기" 같은 SKU 가 CSV 에서 [제외 의견 TRUE] 로 마킹된 이유와 동일.

## 2. 선물 컨텍스트 5분류

사용자가 정리한 카테고리를 정제해, 카카오 선물하기 상에서 **상품 분류 탭으로 노출 가능한 5가지 컨텍스트**로 묶었다. 각 컨텍스트는 상품 진열에서 한 묶음으로 묶이고, 카카오 측 분류/노출 협상에도 사용 가능하다.

| 컨텍스트 | 트리거 | 주는 가치 | 받는 사람의 반응 |
|---------|-------|----------|----------------|
| **C1. 인생 이벤트 축하/축복** | 결혼, 출산/임신, 이직·승진, 합격, 이사 | 축복 + 조언 약간 | "고마워, 도움이 될 것 같아" |
| **C2. 응원/위로** | 취준, 시험, 진로 고민, 다이어트 | 격려 + 조언 약간 | "약해질 뻔했는데 힘이 됐어" |
| **C3. 시즈널 (연중 정해진 시점)** | 신년, 입학 | 한 해/시기 가이드 | "마침 궁금했는데 잘 받았어" |
| **C4. 가벼운 재미 / 깔아두기** | 친구끼리, 생일 인사, 기프티콘 대체 | 호기심·재미 | "킬킬, 재미있다" |
| **C5. 셀프 (나에게 선물)** | 임신 준비, 다이어트 결심, 새해 다짐 | 본인 동기부여 | (구매자=받는자) |

> 사용자가 노트에 적은 "초반 10개는 카카오 선물하기에 오는 사용자에게 끌리는 상품" = **C1 + C3 + C4 가 메인 진열, C2 는 보조, C5 는 깔아두기**.

```
[일반 응원·생일]
  범용 1      → KS00016 ₩19,500   인생역전 3대운
─────────────────────────────────────────────────
[재미·깔아두기]
  재미 1      → KS00015 ₩17,000   평생 재물
  재미 2      → KS00017 ₩17,000   30년 뒤 재물
─────────────────────────────────────────────────
[시즈널 (11~1월 부스트)]
  신년        → KS00001 ₩33,500   2026 신년운세
─────────────────────────────────────────────────
```

**가격대 분포**: ₩16,500 ~ ₩33,500. 신년(₩33,500) 외에는 ₩16,500~₩22,500 구간에 9개가 몰려 있어 카카오 선물하기 가격 부담 구간(2~3만원대)과 정합.

## 5. 상품명 / 노출 메시지 타입 권장

CSV 에 4가지 메시지 타입(페르소나 지목형, 메시지형, 태그형, 룰베이스AI)이 준비되어 있다. **선물 컨텍스트별로 메시지 톤이 달라야 한다**.

| 컨텍스트 | 권장 메시지 타입 | 이유 |
|---------|---------------|------|
| **C1 인생 이벤트 축하** (결혼/이직/이사/합격) | **메시지형** | 받는 사람을 향한 축복·기원 톤. 부담 없이 받음. 예: "백년해로 꽃길만 걷는 부부 되길 💐" |
| **C2 응원** (관운/짝사랑 속마음) | **페르소나형 (조심)** | "사직서 품은 너에게"식 페르소나는 친한 사이만. 1차 출시 시 메시지형으로 톤다운 권장 |
| **C3 신년 시즈널** | **메시지형 + 태그형** | 신년 인사처럼 통째로 받음. 태그형은 검색 노출용 |
| **C4 재미/깔아두기** | **페르소나형** | "얼빠 본능 숨기지 못하는 너에게"식 깔깔이 톤이 친구끼리 잘 먹힘 |

### 1차 출시 10개 메시지 타입 매핑 안

| KS | 권장 타입 | 채택 사유 |
|----|----------|----------|
| KS00013 결혼 궁합 | **메시지형** | "백년해로" 축복 톤 — 예비 부부에게 부담 없음 |
| KS00022 직업 컨설팅 | **메시지형** | "적성에 찰떡인 천직" 톤. 페르소나("잠 못 드는 너")는 친한 사이만 |
| KS00018 3번의 기회 | **메시지형** | "찾아올 대박 기회 절대 놓치지 않길" — 응원·합격·생일 범용 |
| KS00020 운 트이는 동네 | **메시지형** | "발길 닿는 곳마다 돈방석" — 집들이 부드러움 |
| KS00016 인생역전 3대운 | **메시지형** | "인생 전성기" — 생일·응원 범용 |
| KS00011 사주 궁합 | **페르소나형** | "그 사람에게 푹 빠진 너에게" — 커플 친구에게 가벼움 |
| KS00012 3가지 궁합 | **메시지형** | 진심 선물 라인이라 축복 톤이 더 어울림 |
| KS00001 2026 신년운세 | **메시지형** | "올 한 해 대박 나길 응원해" — 신년 인사용 |
| KS00015 평생 재물 | **페르소나형** | "영앤리치 꿈꾸는 친구" — 깔깔이 |
| KS00017 30년 뒤 재물 | **페르소나형** | "재테크 진심" — 깔깔이 |

> 룰베이스AI / 태그형은 1차 출시 후 카카오 노출 매대 협상 결과 보고 보강 (검색 노출, 카카오 자체 추천 알고리즘 활용용).

## 6. 섬네일 디자인 가이드 — 컨텍스트별 톤

> 본격 디자인은 `/design` 에서 처리. 분석 단계에서는 **컨텍스트별로 묶을 톤 가이드**만 제시.

| 컨텍스트 | 톤 키워드 | 색감 | 받는 사람 인상 |
|---------|----------|------|---------------|
| **C1 축하/축복** | 단정함·고급감 | 골드, 네이비, 화이트 + 한국적 패턴 (꽃·구름) | "축의금 같은 정중함" |
| **C2 응원** | 밝음·따뜻함 | 옐로, 코랄, 라이트 그린 | "기프티콘처럼 가벼움" |
| **C3 신년** | 시즌감 강조 | 새해 컬러(레드, 골드) + 띠 모티프(말 등) | "한 해 시작" |
| **C4 재미** | 발랄·디지털 | 비비드, 그라데이션, 이모지 활용 | "인스타 스토리감" |

특히 결혼·이직·이사 같은 C1 SKU 는 **현재 헬로우봇 인앱 섬네일을 그대로 쓰면 안 된다** — 인앱 섬네일은 사용자가 자기 운세 보러 들어온 톤(자극적 페르소나)이라 카카오 선물하기에서는 "내가 이걸 받았다고?" 위화감 발생. 카카오 선물하기 전용 섬네일 세트 필요.

## 7. 다음 단계 / Open Questions

### 7.1 의사결정이 필요한 항목 (사용자/PO 확인)

1. **출시 일정 vs 시즈널 시점**: 1차 출시 시점이 2026년 6~7월이라면 KS00001 신년운세는 "사전 등록 후 11월 푸시" 전략으로 가는 게 효율적. 1차 10개에서 신년 SKU 를 빼고 다른 C2 응원 SKU 를 채울지 검토 필요.
2. **출산 케이스 1차 포함 여부**: KS00023 자식복을 메시지 톤 다듬어서 1차에 합류시킬지, T2 로 미룰지.
3. **커플 SKU 1개로 압축할지**: KS00011·12 는 동일 콘텐츠 다른 가격대. 카카오 매대에서 한 카테고리에 2개 노출이 자기잠식 우려라면 KS00012 만 진열.
4. **신년 SKU 묶음**: KS00001·02·03 을 한 매대에 묶어 노출할지 (정통/마니아/애정운).

### 7.2 추가 분석이 필요한 항목 (데이터)

5. **앱 내 매출 데이터로 검증**: 위 10개가 헬로우봇 앱 내에서 어떤 매출 순위인지 BigQuery 로 교차 검증 (`/dev-data`). "친구가 좋아할 만한 스킬" 가설이 자기소비 데이터와 일치하는지.
6. **카카오 선물하기 동종 카테고리 가격 분포**: 운세/타로 외 다른 디지털 콘텐츠 교환권(웹툰·웹소설·이모티콘)의 단가 분포를 보고 ₩16,500~₩33,500 가격대가 시장에 맞는지 검증.

### 7.3 후속 작업 (확정 후)

7. 선정된 10개에 대해 **카카오 매대 노출용 섬네일 디자인 발주** (`/design`)
8. 선정된 10개의 **상품 정보 입력 시트 작성** (이미 진행 중인 `product-submission-draft-v1.xlsx` 갱신)
9. 시즈널 SKU 의 **사전 등록 + 노출 ON/OFF 운영 계획**

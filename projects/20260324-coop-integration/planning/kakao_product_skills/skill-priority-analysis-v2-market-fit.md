## 0. v1 → v2 무엇이 달라졌나

| | v1 (2026-05-14 오전) | v2 (본 문서) |
|---|---|---|
| **분석 렌즈** | 사용자 컨텍스트 5분류(C1~C5)에 SKU 골고루 배치 | **시장 적합성** + **즉시 이해도** + **전 연령(10~40대)** + **코어 팬 매력 유지** |
| **선정 동기** | 컨텍스트 커버리지 (각 진열대에 SKU 1개씩) | 카카오 선물하기 매대에서 **이미 검증된 사주·운세 굿즈 카테고리의 행동 패턴**과 가장 정합한 SKU 우선 |
| **타깃 폭** | 30대 여성 1차 + 2030 남성·40대 보조 | **10대~40대 전 연령 커버**가 명시 요구사항 |
| **코어 팬 고려** | 컨텍스트에 따라 자동으로 배제 | **사주·운세를 좋아하는 사람이 매대에서 "오 좋아"라고 반응하는가**를 별도 가중치로 평가 |

## 5. 의사결정이 필요한 항목 (사용자 확인)

### 5.1 KS00029 가격·구성 확정

CSV에서 KS00029 가격·메시지 타입 모두 비어있음. 1차 진입 SKU로 결정하려면 다음 확인 필요:

- [ ] 가격대 결정 (다른 SKU 평균 ₩19,500 vs 학부모용 프리미엄)
- [ ] 메시지형 카피 작성 ("우리 아이의 빛나는 재능 꼭 발견하길" 톤)
- [ ] KS00024 "내 아이는 출세할 사주일까?"와의 콘텐츠 중복 정리 (둘 중 하나만 노출 권장 — KS00029 우위)

### 5.2 신년 시즈널 SKU 운영 방식

KS00001은 시즈널이지만 출시 시점이 6~7월이라면:
- **옵션 A**: 1차 출시에 포함, 시즌 외에는 매대 노출 OFF, 11월부터 부스트
- **옵션 B**: 1차에서 제외하고 11월에 별도 출시 — 그 자리에 KS00017(30년 뒤 재물)을 채워 재물 라인 강화

권장: **옵션 A** — 카카오 매대 등록 행정 비용 고려 시 미리 등록해두고 노출 ON/OFF로 운영.

### 5.3 커플 SKU 1개 압축 vs 2개 라인업 유지

KS00011·12는 동일 콘텐츠 다른 가격대. 카카오 매대에서 자기잠식 우려와 가격 라인업 다양성 사이 트레이드오프:
- **2개 유지**: 라이트(₩19,500) / 진심(₩22,500) — v2 권장
- **1개 압축**: KS00012만 (진심 톤이 선물에 더 적합)

### 5.4 메시지 타입 일괄 메시지형 채택 확인

CSV 4가지 타입 중 매대 노출용은 메시지형으로 통일 ([§3.3 M2 검증](#m2-즉시-이해도-헬로우봇-모르는-사용자도-살-만한가)). 사용자 동의 시 1차 10개 모두 메시지형 카피로 확정.

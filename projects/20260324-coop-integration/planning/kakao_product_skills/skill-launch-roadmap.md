# 카카오 선물하기 스킬 출시 로드맵 (v3)

> **문서 성격**: 4개 판단기준에 차근차근 답하면서 최종 출시 라인업·스케줄을 도출하는 종합 리포트.
> **방법**: 각 기준마다 (질문 의도 → 가설 매트릭스 → 항목별 결정 → 결과)의 4단계로 진행.
> **상태**: 1번 기준 완료 / 2~4번 진행 예정.
> **검증 마크**: `🔍 [데이터 검증 필요]` 표시 항목은 BigQuery·외부 자료로 확정 필요.

- ⚠ **46-50 타로 33.8% 재상승 (실측 보정)** — 기존 가설에서 보지 못한 시그널. 가설: 자녀(20대)·손주 세대 영향, 또는 결제층 변화. 추가 분석 필요하나 1차 라인업에는 영향 없음 (40대 SKU는 자녀·부모 발신 위주).
- ⚠ **APP 전반 타로 비중이 가설보다 높음** — A.7.2 13-25세 평균 타로 30% 이상. A.2의 "타로" 마크가 적었던 것은 사주 중심 가설의 편향. 카카오 라인업 검토 시 **타로 SKU 비중을 가설보다 더 확보** 검토 가치 있음.

## 2. 2번 기준: 헬로우봇 모르는 카카오톡 사용자도 구매하여 선물하고 싶은 순

### 2.1 질문의 의도

```
[헬로우봇 브랜드 인지 = 0]  +  [카카오 선물하기 노출만 보고 5초 안에 결정]
                              ↓
        "이걸 친구한테 보내야지"가 즉시 떠오르는 SKU 순서
```

리포트A 시사점 1·3·5와 정확히 매핑. **"본인이 헬로우봇을 안 써봤어도, 친구에게 보낼 만하다고 느껴지는 정도"**.

### 2.2 비유저 매력도 5축

| 축 | 의미 | 약하면 |
|---|---|---|
| ① 상황 명확성 | 어떤 상황에 보내는지 즉시 매핑 | 어떤 친구한테 보낼지 안 떠오름 |
| ② 메시지 보편성 | 받는 사람 부담·오해 없음 | "이걸 왜 줬지?" 거부감 |

| ④ 가격 부담 | 1~3만원 안에 위치 | 결제 직전 이탈 |
| ⑤ 개인화 부담 | 사주 입력 부담 (운세 공통 약점) | 받은 사람이 안 씀 |

## 4. 4번 기준: 코어 팬이 매력 못 느끼는 것 회피

### 4.1 질문의 의도

```
[운세 코어 팬 = 사주·운세 콘텐츠 헤비 사용자]
  = 헬봇 헤비유저 + 외부 운세 앱 헤비 + 카카오 운세 굿즈 구매자

[매력 못 느낌 = 가벼움·통속·왜곡으로 외면]
  → 라인업 전체의 "톤 일관성" 훼손 → 헬봇 브랜드 정통성 자산 소모
```

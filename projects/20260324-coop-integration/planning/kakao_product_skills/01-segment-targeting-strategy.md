## 0. 토론 진행 방식

## 2. C1 — 핵심 코어 (26-35 여성 중심)

### 2.1 26-30 — "사회 정착·라이프이벤트 폭발기"

`[데이터 시그널]`
- 헬봇 APP 매출 점유 **31.4%** (단일 세그먼트 최대)
- pay_rate 25.5% / ARPPU 21,404원 / 신규 결제율 31.0%
- 사주 78.8% / 타로 17.7% / 30K+ Whale 4.2%
- A.4: 카카오 30대 구매율 여성 77% (최대 큰손)
- A.5: "30대 여성 Primary — 라이프이벤트 밀집, 5~30만원 가격대"

`[페르소나 요약]`
- 대학 졸업~결혼 적령기. 친구·연인·동료·부모 전 방향 발신.
- "받고 싶은 위시리스트" + "이미 받아본 경험" 둘 다 있음.
- 사주 콘텐츠 자가소비 빈도 압도적, 동시에 친구한테 보내고 싶음.
- 키워드: **결혼·이직·이사·여행·자기 보상**

`[공략 방향]`

| 레버 | 방향 |
|---|---|
| **SKU 컨셉** | 라이프이벤트 축하(결혼·이직·이사·합격) + 친구 라이트 응원 + 셀프 보상 패키지 |
| **카테고리** | 궁합·결혼 사주 / 직업 컨설팅 / 이사 풍수 / 종합 사주 보고서 |
| **가격대** | ₩17,000~₩29,900 (메인) + ₩9,900 라이트 (친구 깔깔이용) |
| **메시지 톤** | 메시지형 ("결혼 축하해, 둘의 사주가 어떻게 어울리는지") — 본인 가치 + 친구 응원 둘 다 |
| **노출 레버** | L1 라인업 두텁게 + L3 14개 선물테마 전 매핑 + L5 시즌 캠페인 (결혼 5월, 이사 봄·가을) |
| **시즌 부스트** | 결혼 시즌 (3~6월, 9~11월), 신년, 어버이날(부모 발신), 친구 생일 (연중) |

`[트레이드오프]`
- 26-30 단일 세그먼트에 라인업 너무 집중 시 다른 세그먼트 매대 점유율 손실 가능
- 라이프이벤트 매대(이사·결혼)는 카카오 알고리즘이 운세 SKU에 노출 가산점 줄지 미정 → MD 협상 의존
- 신규 결제율 31%이지만 ARPPU 21K 수준 — LTV 확장은 재구매·재발신 K값에 달림

`[PM 1차 입장]` **1차 라인업 핵심 60% (8/14 SKU)는 26-30 타깃**. 결혼·이직·이사·합격·궁합·범용 응원·재미·범용 종합 — 모두 매핑 가능. 단 가격대를 ₩17,000~₩22,500 / ₩9,900 두 라인으로 분리.

`[사용자 결정 필요]`
- 26-30 라이트 SKU(₩9,900) 라인 신설 의지 (현재 28개에 없음)
- 셀프 보상 SKU(자기 자신에게 주는 카드) 노출 정책 — 카카오 선물하기에 직접 노출 vs 카톡 채널·인앱에서만

### 2.2 31-35 — "결혼·출산·중간관리자 진입기"

`[데이터 시그널]`
- 헬봇 APP 매출 점유 **24.7%** (단일 2위)
- pay_rate **25.8%** (최고) / ARPPU **23,534원 (최고)** / 신규 결제율 **39.4% (최고)**
- 사주 77.2% / 타로 16.9% / 30K+ Whale **4.6%** (최고급)
- A.4: 30대 구매율 여성 77%
- A.5: "30대 여성 Primary" + A.6: "이직·결혼 → 종합 사주 보고서 ₩29,900"

`[페르소나 요약]`
- 결혼·출산·승진의 절정. 배우자·부모·자녀·동료 모두 발신.
- 자기 운세는 이미 익숙, **종합 보고서·프리미엄 SKU** 수용도 높음.
- 자녀(0~5세) 임신·출산 키워드 진입.
- 키워드: **부부 궁합·임신·중간관리자 이직·자녀 시작·부모 효도**

`[공략 방향]`

| 레버 | 방향 |
|---|---|
| **SKU 컨셉** | 부부 궁합 / 임신·출산 / 종합 사주 보고서 / 부모 효도 (어버이날·환갑) / 이직·승진 |
| **카테고리** | 결혼 사주 / 임신운 / 종합 보고서 / 부모용 신년·건강 / 직업·재물 |
| **가격대** | **₩22,500~₩49,900 (프리미엄 라인)** + 셀프 종합 패키지 ₩39,900 |
| **메시지 톤** | 메시지형 (진심 톤) — "함께한 1년, 우리의 사주는 어떤 사이일까요" |
| **노출 레버** | L1 프리미엄 SKU 라인 강조 + L4 카카오 단독 SKU 협상(자사몰 차별 패키지) + L5 어버이날·결혼시즌 캠페인 |
| **시즌 부스트** | 어버이날(5월), 결혼 기념일(연중), 임신·출산 부스트, 신년 |

`[트레이드오프]`
- 프리미엄 SKU 객단가 매력적이나 인지·결제 결정 시간 길어짐 — 매대 클릭 후 구매율 측정 필수
- 임신·출산 SKU는 난임·미혼 응원 톤으로 받는 사람 부담 우려 → 메시지형 톤 정교화 필요
- 부모 효도 SKU는 받는 사람(50대+)의 디지털 콘텐츠 사용성이 변수 — 종이 인쇄 PDF 동봉 옵션 검토

`[PM 1차 입장]` **31-35는 ARPPU·Whale 모두 최고치 → 프리미엄 SKU 라인의 메인 타깃**. 1차 라인업에 ₩29,900~₩39,900 종합 보고서 1~2개 + ₩22,500 진심 궁합 1개 = 3~4개 SKU 직접 매핑. C1 코어 중에서도 **이 세그먼트가 최대 ARR 잠재력**.

`[사용자 결정 필요]`
- 임신·출산 SKU(KS00025·26·23 류) 1차 진입 여부 — 메시지 톤 톤다운 가능한지
- 카카오 단독 프리미엄 SKU 신규 기획 의지 (자사몰 ₩39,900 → 카카오 ₩29,900 전용 패키지)

## 3. C2 — 구매력 세그먼트 (36-50)

### 3.1 36-40 — "가족 중심·정통 사주 정점"

`[데이터 시그널]`
- 헬봇 APP 매출 점유 11.4%
- pay_rate 24.5% / ARPPU 23,691원 / 신규 결제율 34.7%
- 사주 **81.7% (전 연령 최고)** / 타로 14.1%
- 30K+ Whale **5.4% (최고)**
- A.4: 40대 구매율 여성 58% (성별차 큼 — 남성 14%)
- A.5: "40대 여성 — 가족 중심, 5~30만원 가격대"

`[페르소나 요약]`
- 결혼·출산 후 본격 육아·가족 운영기. 자녀(초등·중등)·배우자·부모 중심 발신.
- 카카오 선물하기 사용 줄어드는 시점이지만 **사용 시 객단가 크고 정통 사주 신뢰**.
- 사주 81.7%는 정통 명리·풍수 콘텐츠가 진가를 발휘하는 세그먼트.
- 키워드: **자녀 초등·중등 / 부모 명절·환갑 / 가족 건강 / 자가 마련**

`[공략 방향]`

| 레버 | 방향 |
|---|---|
| **SKU 컨셉** | 자녀 사주 / 가족 종합운 / 부모 효도 / 부부 미래 / 자가·재물 |
| **카테고리** | 자녀운 / 가족 종합 / 부모용 신년·건강 / 재물 / 풍수 |
| **가격대** | **₩29,900~₩49,900 (정통 프리미엄)** — Whale 5.4% 흡수 |
| **메시지 톤** | 메시지형 (정중·진심) — "아이의 빛나는 재능, 1년의 길을 미리 봐드려요" |
| **노출 레버** | L1 정통 사주 라인 + L5 어버이날·환갑·이사 캠페인 + L4 카카오 단독 SKU (자녀 사주 등) |
| **시즌 부스트** | 어버이날(5월) / 자녀 초등 입학(1~3월) / 명절(추석·설) / 신년 |

`[트레이드오프]`
- 카카오 30대 → 40대 활동 감소 (구매율 77% → 58%) → 절대 거래액 감소 가능
- 40대 남성 14% — 거의 발신 안 함 → 발신자 풀이 좁음 (여성 중심)
- 자녀 사주 SKU는 종교·가치관 거부감 가능 → 메시지 톤 신중

`[PM 1차 입장]` **정통·프리미엄 라인의 핵심 타깃**. KS00029(학부모) 이미 1차 라인업, 추가로 가족 종합·부모 효도 SKU 1~2개 기획 검토. **객단가 ₩29,900+ 라인이 36-40에 정합** — Whale 5.4% 자산 활용.

`[사용자 결정 필요]`
- 자녀 사주 라인(초등 입학·진로) 확장 의지 (KS00029 + 신규 SKU 1~2개)
- 명절(추석·설) SKU 신설 의지

### 3.2 41-45 — "학부모·중간관리자 정착기"

`[데이터 시그널]`
- 헬봇 APP 매출 점유 4.9% / pay_rate 24.6% / ARPPU 23,248원
- 사주 75.5% / 타로 15.9%

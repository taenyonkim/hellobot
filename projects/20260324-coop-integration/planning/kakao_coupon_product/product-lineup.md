## 2. 스킬 교환권 (선정 필요)

> 입력 자료: 전체 스킬 목록 + 가격/판매 실적 — `헬로우봇 카테고리별 스킬 목록 - 20260408.xlsx`
> 쿠프 요청(1pager): **"지금 인기 있는 스킬이에요" 영역 단일 콘텐츠 상품 + 헬로우봇 제안 수요 높은 콘텐츠**

### 2.1 선정 기준 (합의 필요)

- [ ] 기준 A: **메인 페이지 "지금 인기 있는 스킬" 노출** 상품 우선
- [ ] 기준 B: **ARPPU / 구매 전환율 상위** 상품
- [ ] 기준 C: **단일 콘텐츠(fixed_menu_seq 하나로 매핑 가능)** 상품
- [ ] 기준 D: **가격대 분포** — 저가(1만원대) / 중가(2만원대) / 고가(3만원 이상) 균형
- [ ] 기준 E: **시즌/한시 상품 제외** (상시 운영 가능한 상품만)

### 2.3 가격 정책 (확정 2026-04-22)

- ✅ **가격 기준**: **웹 판매가 우선** — 웹 판매가가 없으면 웹 정상가 사용
- ✅ **카카오 측 할인 프로모션 대응**: **론칭 시점에는 미적용** (향후 프로모션 시 별도 협의)
- ✅ **가격 범위**: **₩1,500 ~ ₩50,000** (스킬 10종 합산 기준 범위)

### 2.4 100% 할인 쿠폰 스펙 (서버 자동 생성)

각 스킬 교환권 등록 시 발급되는 내부 쿠폰:

- couponSpec: skill 전용, 100% 할인
- 적용 대상: `fixed_menu_seq` 단일
- 유효기간: **무제한** (ISS-021/022 확정)

**확정 사항** (2026-04-22):

- ✅ 앱 내 쿠폰 유효기간: **무제한** (ISS-021/022 sentinel `2099-12-31` + `isUnlimited: true`)
- ✅ 1장당 사용 횟수: **1회 (전액 소진)**

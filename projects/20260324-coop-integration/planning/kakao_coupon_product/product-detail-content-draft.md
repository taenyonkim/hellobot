## 0. 카피 작성 원칙

- **공통 영역(STEP 01~04)은 두 상품 모두 동일 카피 사용** — 사용자 학습 비용 최소화
- **상품별 차별 영역(STEP 05~)** 만 분리 — 하트는 즉시 충전, 스킬은 이용권 발급 후 사용
- 톤: **존댓말 + 짧은 문장** (SNOW 톤 동일)
- 헬로우봇 명칭은 **`헬로우봇`** 으로 통일 (앱/서비스 모두)
- 화면 캡처는 별도 디자인 작업 — 본 문서는 카피만 정의

## 1. 공통 카피 (하트 충전권 / 스킬 교환권 동일)

### 1.1 헤더 영역 (상품명 위 카피)

> 상품 카드 상단에 들어갈 짧은 캐치프레이즈

```
나의 마음을 톡, 운세를 선물하세요
하루의 안부를 묻는 가장 다정한 방법, 헬로우봇
```

### 1.2 쿠폰 등록 방법 (공통 STEP)

> SNOW의 “구독권 등록 방법” 섹션 대응. **두 상품 모두 동일** STEP 01~04 사용.

#### 쿠폰 등록 방법

선물 받은 쿠폰 번호를 헬로우봇에서 등록하면 바로 사용할 수 있어요.

| STEP | 카피 | 화면 캡처 안내 |
| --- | --- | --- |
| **STEP 01** | 헬로우봇 앱을 실행해 주세요. | 앱 아이콘 / 스플래시 |
| **STEP 02** | 회원가입 또는 로그인을 진행해 주세요.<br/>(카카오 · 네이버 · 애플 · 구글 등 소셜 로그인 가능) | 로그인 화면 |
| **STEP 03** | 하단 [프로필] 탭에서 [쿠폰]을 눌러 주세요. | 프로필 화면 → 쿠폰 진입 |
| **STEP 04** | 쿠폰 번호를 입력하고 [등록] 버튼을 눌러 주세요. | 쿠폰 입력 화면 |

> 💡 **헬로우봇 앱이 없다면?** 구글 플레이스토어 · 앱스토어에서 “헬로우봇”을 검색해 설치하실 수 있어요.

### 1.3 STEP 05 분기 안내 (상품별 분기 라벨)

```
※ 등록한 쿠폰 종류에 따라 다음 단계가 달라요.
- 하트 충전권 → STEP 05 (하트 충전권 사용 방법) 참고
- 스킬 교환권 → STEP 05~06 (스킬 교환권 사용 방법) 참고
```

## 2. 하트 충전권 — 상품별 카피

### 2.1 상품 소개 영역 (상품 카드 상단)

```
헬로우봇 하트 {N}개 충전권

🤍 등록 즉시 하트가 내 계정으로 충전돼요
🤍 다양한 사주·타로·심리 콘텐츠 이용에 사용할 수 있어요
🤍 유효기간 없이 언제든 사용 가능해요
```

> {N}은 상품 SKU별 변경 — 25 / 60 / 210 / 360 (`product-lineup.md` §1)

### 2.2 등록 방법 — 하트 충전권

> §1.2 STEP 01~04를 동일하게 노출한 후 STEP 05 추가

| STEP | 카피 |
| --- | --- |
| STEP 01 | (공통) 헬로우봇 앱을 실행해 주세요. |
| STEP 02 | (공통) 회원가입 또는 로그인을 진행해 주세요. |
| STEP 03 | (공통) [프로필] → [쿠폰]을 눌러 주세요. |
| STEP 04 | (공통) 쿠폰 번호를 입력하고 [등록]을 눌러 주세요. |
| **STEP 05** | **하트가 즉시 충전됩니다. [프로필] 탭에서 충전된 하트를 확인할 수 있어요.** |

> **사용 방법**: 하트는 헬로우봇의 모든 사주·타로·심리 콘텐츠 결제에 자유롭게 사용할 수 있어요. 구매하고 싶은 콘텐츠를 선택한 뒤, 하트로 결제해 주세요.

## 3. 스킬 교환권 — 상품별 카피

### 3.1 상품 소개 영역 (상품 카드 상단)

```
{스킬명} 이용권

✨ 등록 즉시 [내 쿠폰함]에 이용권이 추가돼요
✨ 이용권을 누르면 챗봇이 바로 운세를 알려드려요
✨ 유효기간 없이 언제든 사용 가능해요
```

> {스킬명}은 상품 SKU별 변경 — KS00001~KS00010 (`product-lineup.md` §2)

### 3.2 등록 방법 — 스킬 교환권

> §1.2 STEP 01~04를 동일하게 노출한 후 STEP 05~06 추가

| STEP | 카피 |
| --- | --- |
| STEP 01 | (공통) 헬로우봇 앱을 실행해 주세요. |
| STEP 02 | (공통) 회원가입 또는 로그인을 진행해 주세요. |
| STEP 03 | (공통) [프로필] → [쿠폰]을 눌러 주세요. |
| STEP 04 | (공통) 쿠폰 번호를 입력하고 [등록]을 눌러 주세요. |
| **STEP 05** | **[쿠폰] 목록 맨 위에 ‘이용권’ 카드가 추가돼요.** |
| **STEP 06** | **이용권 카드를 누르면 해당 스킬 상세 화면으로 이동해요. [지금 보러가기]를 누르면 챗봇이 바로 운세를 시작해요.** |

### 3.3 사용 방법 — 스킬 교환권 (별도 섹션)

> SNOW 사례의 “NEW AI 프로필 사용 방법” 형태로 STEP 분리. 스킬 교환권은 “등록 → 사용” 두 단계가 직관적이지 않아 별도 섹션을 권장.

#### 스킬 교환권 사용 방법

| STEP | 카피 |
| --- | --- |
| STEP 01 | [프로필] → [쿠폰] → 등록한 ‘이용권’ 카드를 눌러 주세요. |
| STEP 02 | 스킬 상세 화면에서 [지금 보러가기]를 눌러 주세요. |
| STEP 03 | 챗봇이 자동으로 운세를 시작합니다. 하트 차감 없이 무료로 이용하실 수 있어요. |

## 4. 유의사항 (공통)

> 두 상품 모두 동일 카피. 푸터 영역에 작은 글씨로 노출.

```
[유의사항]

· 본 상품은 헬로우봇 모바일 앱(iOS · Android)에서 등록할 수 있어요. 웹에서도 등록 가능하며, 모든 채널에서 동일한 계정에 적립됩니다.
· 쿠폰 등록 시점에 헬로우봇 회원가입 또는 로그인이 필요해요.
· 등록한 쿠폰은 등록한 계정에서만 사용 가능하며, 다른 계정으로 양도하거나 분할 사용할 수 없어요.
· 한 장의 쿠폰은 1회만 등록 가능하며, 등록 후에는 카카오 선물하기에서의 환불·교환이 제한될 수 있어요. (환불 정책은 카카오 선물하기 정책을 따릅니다.)
· 등록 후 적립된 하트와 발급된 이용권의 유효기간은 별도 표기가 없는 한 무제한으로 운영돼요. 단, 헬로우봇 서비스 종료 시에는 사용이 제한될 수 있어요.
· 스킬 교환권은 카카오 선물하기 상품 상세에 표기된 단일 스킬에 한해 사용할 수 있으며, 다른 스킬로 교환·환불되지 않아요.
· 스킬 교환권 1장당 해당 스킬을 1회 무료로 이용할 수 있어요.
· 카카오 선물하기에서 발급된 쿠폰의 사용 기한은 발급일로부터 카카오 선물하기 정책을 따릅니다. 기한 내 헬로우봇 앱에서 등록해 주세요.
· 결제 취소·반품 등으로 인해 쿠폰이 무효화된 경우 헬로우봇에서도 사용이 제한됩니다.
· 만 14세 미만 고객은 법정대리인의 동의 후 사용해 주세요.
· 본 상품은 부정 사용·매크로 등 비정상적인 사용이 확인될 경우 회수될 수 있어요.

[고객센터 안내]
· 헬로우봇 등록·사용 관련 문의: 헬로우봇 앱 [프로필 → 고객센터] 또는 help@hellobot.co
· 결제·환불·발송 관련 문의: 카카오 선물하기 고객센터
```

## 5. 사용 가이드 (작업자용)

### 5.1 SKU별 매핑

| 상품 코드 | 상품 카피 사용처 |
| --- | --- |
| `KH00001` ~ `KH00004` | §2 하트 충전권 카피, {N} 자리 SKU별 치환 |
| `KS00001` ~ `KS00010` | §3 스킬 교환권 카피, {스킬명} 자리 SKU별 치환 |

### 5.2 카카오 선물하기 입점 시 노출 순서 (제안)

1. 상품 소개 (§2.1 또는 §3.1)
2. 쿠폰 등록 방법 (§1.2 + §2.2 또는 §3.2)
3. 스킬 사용 방법 (§3.3, 스킬 교환권 한정)
4. 유의사항 (§4)
5. 고객센터 안내 (§4 하단)

### 5.3 카피 확정 전 검토 필요 항목

- [ ] 쿠프마케팅 측 환불 정책 카피 검수 (§4 “등록 후 환불 제한” 문구)
- [ ] 카카오 선물하기 측 입점 가이드 — 유의사항 글자 수/형식 제약 확인
- [ ] 헬로우봇 고객센터 이메일/메뉴 확정 (`help@hellobot.co` 임시 표기, 운영팀 확인 필요)

- [ ] 카카오/쿠프 상품권 자체 유효기간 정책 회신 후 §4 카피 보강 (`product-lineup.md` §1 하단 메모 반영)

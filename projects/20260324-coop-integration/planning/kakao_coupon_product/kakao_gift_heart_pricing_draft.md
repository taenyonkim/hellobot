# 카카오 선물하기 하트 상품 가격 (초안)

## 1. 상품 구성

| 상품명 | 판매가 | 하트 수 | 하트당 단가 | 정가 대비 할인율 |
|--------|--------|---------|------------|----------------|
| 하트 25 | ₩5,000 | 25 | ₩200 | 17% |
| 하트 60 | ₩10,000 | 60 | ₩167 | 31% |
| 하트 210 | ₩30,000 | 210 | ₩143 | 40% |
| 하트 360 | ₩50,000 | 360 | ₩139 | 42% |

> 정가 기준: 하트당 ₩240 (웹 정가 기준)

---

## 2. 가격 산정 근거

### 2.1 기준: 웹 판매가

카카오 선물하기는 웹 결제와 동일하게 앱스토어 수수료가 없는 채널이므로, **웹 판매 단가**를 기준으로 산정.

### 2.2 기존 웹 가격 커브와의 정합성

기존 웹 상품의 하트당 단가 커브에 카카오 상품을 매핑하면:

```
하트당 단가 (₩)
240 ┤ ■ 정가
200 ┤ ● 웹 25하트 ─────────── ★ 카카오 5천원 (25하트, ₩200)
172 ┤ ● 웹 50하트
167 ┤ ● 웹 75하트 ─────────── ★ 카카오 1만원 (60하트, ₩167)
152 ┤ ● 웹 100하트
144 ┤ ● 웹 150~200하트 ────── ★ 카카오 3만원 (210하트, ₩143)
142 ┤ ● 웹 250하트
140 ┤ ● 웹 350하트 ─────────── ★ 카카오 5만원 (360하트, ₩139)
125 ┤ ● 웹 610하트
    └──────────────────────────────────
         25  50  75 100 150 200 250 350 610  하트 수
```

각 카카오 상품이 기존 웹 단가 커브 위에 자연스럽게 위치하여, 기존 사용자가 봐도 가격 위화감이 없음.

### 2.3 상품별 산정 상세

**5천원 상품 (25하트)**
- 웹 하트 25 상품과 동일 (₩5,000 / 25 = ₩200/하트)
- 기존 최소 구매 단위와 일치하여 진입 장벽 낮음

**1만원 상품 (60하트)**
- 웹 50하트(₩172)~75하트(₩167) 구간
- ₩10,000 ÷ ₩167 = 59.9 → **60하트** (₩167/하트)
- 웹 75하트 단가와 정확히 일치

**3만원 상품 (210하트)**
- 웹 150~200하트(₩144) 구간
- ₩30,000 ÷ ₩144 = 208.3 → **210하트** (₩143/하트)
- 웹 150~200하트 단가와 거의 동일 (₩1 차이)

**5만원 상품 (360하트)**
- 웹 350하트(₩140) 구간
- ₩50,000 ÷ ₩140 = 357.1 → **360하트** (₩139/하트)
- 웹 350하트 단가와 거의 동일 (₩1 차이)

### 2.4 단가 체감 구조

| 구간 | 하트당 단가 | 전 구간 대비 |
|------|------------|-------------|
| 5천원 → 1만원 | ₩200 → ₩167 | -17% |
| 1만원 → 3만원 | ₩167 → ₩143 | -14% |
| 3만원 → 5만원 | ₩143 → ₩139 | -3% |

- 저가 구간에서 단가 차이가 크고, 고가 구간에서 수렴하는 기존 할인 커브 패턴 유지
- 큰 상품을 선물할수록 이득이라는 인식을 줄 수 있음

## 3. 참고사항

- 카카오 선물하기 수수료율에 따라 하트 수 조정 필요할 수 있음
- 카카오 측 상품 등록 시 가격 제약 조건 확인 필요 (예: 최소/최대 금액, 금액 단위)
- 시즌 프로모션 시 보너스 하트 추가 등 별도 운영 가능
